Projet THE CONVEYOR

Automated Sorting System with NFC & Dolibarr WMS Integration

Description du Projet

THE CONVEYOR automates package sorting using an Arduino-based control system combined with NFC technology. It integrates directly with the Dolibarr Warehouse Management System (WMS) via its REST API to determine package routing based on NFC-tagged product references. Packages are automatically directed to one of three storage warehouses (C, D, or error B) while stock levels and movements are updated in real-time.

Processus de Fonctionnement

  1. Conveyor belt starts, moving a package forward.
  2. Infrared sensor detects the presence of a package.
  3. The conveyor halts temporarily.
  4. If no NFC tag is detected: Servo motor redirects the package to error storage warehouse B.
  5. If NFC tag is detected: UID is read, WMS is queried, and the servo redirects to warehouse C or D accordingly.
  6. The conveyor restarts, delivering the package to its designated path.

Technologies Utilisées

Arduino C/C++ (controller programming), NFC technology (package identification), Dolibarr ERP/CRM (WMS integration), and REST API communication.

Fonctionnalités Clés

  • NFC Reader: Reads product reference data from attached NFC tags.
  • WMS Integration: Communicates with Dolibarr WMS to determine destination and update stock.
  • Automated Sorting: Uses servo motor to direct packages to their assigned warehouse.
  • Error Handling: Redirects untagged packages to an error zone (Warehouse B).
  • Fast Recovery: System resumes normal operations after power loss or manual interruption.

Méthodologie

  1. Hardware integration and controller setup
  2. NFC communication protocol development
  3. Dolibarr API integration and testing
  4. Servo motor control and package routing logic
  5. System validation and performance optimization

Résultats

Successfully developed a fully automated sorting system that reduces manual errors, integrates seamlessly with existing ERP systems, and provides real-time inventory tracking capabilities.

Note: This system enhances efficiency in logistics and warehouse environments by minimizing manual sorting errors, reducing labor costs, and integrating seamlessly with enterprise resource planning (ERP) systems.
