Projet Trash Management

IoT + AR system using Arduino and Unity

Contexte

This IoT project was developed as a miniature prototype using a children's city carpet to simulate a smart city environment. It integrates Arduino-based sensor monitoring with Unity-based Augmented Reality (AR) visualization to create an innovative trash management solution.

Objectifs

  • Monitor the fill level of trash bins in real-time
  • Visualize data through a mobile AR interface
  • Promote smarter waste collection and awareness
  • Create a scalable smart city prototype

Technologies Utilisées

Arduino Uno (sensor integration), Unity 3D (AR visualization), C# scripting, and AR Foundation for mobile deployment.

Méthodologie

  1. Hardware setup with Arduino Uno and 3 sensors
  2. LED integration for visual bin status indicators
  3. Unity AR project creation with serial communication
  4. Real-time data visualization and AR overlay development
  5. Mobile deployment and testing on city carpet prototype

Résultats

Successfully created a functional prototype demonstrating real-time trash bin monitoring with AR visualization. The system effectively tracks fill levels and displays status through intuitive AR overlays.

Axes d'Amélioration

  • Advanced sensor integration for waste classification
  • Machine learning for predictive collection scheduling
  • Expanded city-scale deployment capabilities
  • Enhanced AR user interface and interaction
Note: This project serves as a proof-of-concept for smart city applications, demonstrating the potential for IoT and AR integration in urban waste management systems.
